# Bloque 1: Fundamentos de Python

Ejercicios básicos para repasar los conceptos fundamentales de Python.

## Contenido de la carpeta

- `bucles.py` → Ejemplos de bucles `for` y `while`.
- `condicionales.py` → Ejemplos de condicionales `if`, `elif` y `else`.
- `entrada_salida.py` → Uso de `input` y `print` para interactuar con el usuario.
- `funciones_basicas.py` → Definición y uso de funciones simples.
- `operaciones_basicas.py` → Ejercicios de operaciones aritméticas básicas.
- `variables_basicas.py` → Declaración y manipulación de variables.
